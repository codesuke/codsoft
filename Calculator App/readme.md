# Simple Calculator App

A modern, interactive calculator built with JavaScript, HTML, and CSS. Features include keyboard support, calculation history, sound and theme toggles, and smooth UI effects.

## Features

- **Basic Operations:** Addition, subtraction, multiplication, division, percent, sign toggle, decimal input, and clear.
- **Calculation History:** Displays the latest calculation.
- **Keyboard Support:** Use your keyboard for quick input.
- **Sound Effects:** Toggle sound feedback for button presses and calculations.
- **Theme Switcher:** Easily switch between dark and light modes.
- **Responsive UI:** Subtle animations and effects for a polished experience.
- **Settings Persistence:** Theme and sound preferences are saved in local storage.

## Getting Started

1. **Clone or Download:**

   - Download this repository or copy the files to your local machine.

2. **Run Locally:**
   - Open `index.html` in your browser.

## File Structure

- `index.html` — Main HTML file.
- `style.css` — Calculator styles and themes.
- `script.js` — Calculator logic and interactivity.

## Usage

- Click buttons or use your keyboard to perform calculations.
- Toggle sound and theme using the icons.
- View your latest calculation in the history section.

## Keyboard Shortcuts

- Numbers: `0-9`
- Decimal: `.`
- Operators: `+`, `-`, `*`, `/`
- Equals: `Enter` or `=`
- Clear: `Escape`, `C`, or `c`
- Backspace: `Backspace`

## Customization

- Modify `style.css` for custom themes.
- Extend `script.js` for advanced functions.
